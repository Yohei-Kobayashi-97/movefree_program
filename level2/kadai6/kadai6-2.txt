# 公開されているAPIについて

#　Spotify Web API(音楽データ・プレイヤー API)
##　使用しているAPI概要
    使用する API： Spotify Web API
    提供元： Spotify
    特徴
        楽曲名、アーティスト名、アルバム情報の取得。
        ユーザーのプレイリスト作成や、再生中の楽曲操作が可能。

#　駅すぱあと API（公共交通機関・乗換案内 API）
##　使用しているAPI概要
    使用する API： 駅すぱあと API
    提供元： 株式会社ヴァル研究所
    特徴
        本全国の鉄道、路線バス、高速バス、航空、船のデータを網羅
        運行情報（遅延・運休）の取得
        ICカード運賃と切符運賃
        乗り換え時間/回数など複数提示
